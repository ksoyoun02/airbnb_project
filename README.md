# airbnb_project
------------
### 개인프로젝트
- 프로젝트 명 : airbnb
- 개요 : 여행지 추천 및 숙박예약사이트
- 구현내용 : map api를 사용한 위치표시/
	poi 라이브러리를 사용한 객실등록/ 
	숙박 예약 및 후기작성 등
- 소개 : servlet으로 제작한 프로젝트를
	spring framework에 적용해 구현,
	배운 내용뿐만 아니라 새로운 api,
	라이브러리, 공공데이터를 사용해보았습니다.
